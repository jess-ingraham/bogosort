What is Bogo Sort?

Bogo Sort is sorting algorithm (if it could even be called that). It takes a list of numbers and shuffles them in a random order. Then the algorithm checks the new order for ascending order, if the order is correct the algorithm is finished. However, if the order is incorrect, the list is reshuffled and the process begins again. Because of this, the algorithm tends to be very slow and inefficient.
